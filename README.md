# Linguae Graecae Per Se Illustrata

LGPSI: An open, expansive Greek-reading composition project

## Build an epub file

This requires [pandoc][1].

    $ bash scripts/ebook.sh

## Build PDF file

This requires [pandoc][1], xelatex, and the SBL Greek font.

    $ bash scripts/pdf.sh

[1]: https://pandoc.org/
